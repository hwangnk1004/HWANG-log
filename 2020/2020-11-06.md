#### 네트워크 통신하여 서버로부터 가져온 데이터를 텍스트뷰에 넣기
  
* DecimalFormat 을 이용하여 숫자에 콤마 찍기
	- 자바에서 숫자에 콤마를 찍어 금액을 표기 하는 방법
	```java
		DecimalFormat format = new DecimalFormat("###,###");
	```
	위 같은 패턴 ###,### -> 이는 앞에 세 자리 숫자 + "," + 세 자리 숫자 형태의 패턴을 의미한다.
